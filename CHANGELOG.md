All notable changes to this project will be documented in this file.

## [0.1.0] - 2025-01-01

### Added

- Initial project scaffold with modular node architecture.
- Nine domain node directories: text, image, video, audio, subtitle, consistency, digital_human, export, rag.
- Core framework module with backend abstraction layer.
- CLI entry point (`mosaic`).
- Layered optional dependency groups: video, audio, rag, digital-human, dev.
- Apache-2.0 license.
